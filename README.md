> Предлагаю формат : `%NAME%-%DevUnit%-%Dev|Main%`

### Оформление коммитов
###### [+] Добавление нового функционала
###### [-] Удаление функции
###### [*] Bug fixed
###### [#] TODO
###### [X] Удаление файла
###### [$] Версия кода
### Version Naming System
###### `%v.[0-9].[0-9].[a-e]%`
> `v.0.0.a -> v.0.0.b -> ... -> v.0.0.e -> v.0.1.a -> ... -> v.0.1.e -> v.1.0a -> ... -> v.1.0e`
